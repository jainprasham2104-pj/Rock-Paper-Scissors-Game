# Rock Paper Scissors Game

## Description
A command-line Rock Paper Scissors game developed using Python where the user competes against the computer.

## Features
- User vs Computer gameplay
- Random computer choices
- Score evaluation
- Interactive experience

## Technologies Used
- Python 3
- Random Module

## How to Run
1. Download the project.
2. Open the terminal.
3. Run:

python rock_paper_scissors.py
